## 1. 快速入门

### 1.1 启动应用

```bash
# 激活虚拟环境后
python main.py
```

应用启动后会加载主界面，左侧为导航栏，右侧为动态内容区。

### 1.2 界面布局

- **左侧导航栏**（宽 210px）：11 个功能页面入口
- **右侧内容区**：根据所选页面动态渲染
- **右上角**：Bridge 连接状态指示器
- **底部状态栏**：显示当前操作反馈

## 2. K线图表

### 2.1 查看K线

1. 点击左侧导航「📈 K线图」进入 K 线页面
2. 在顶部搜索框输入股票代码或名称（如 `000001` 或 `平安`），按回车或点击搜索结果
3. 图表自动加载该股票的日线数据

### 2.2 切换周期

点击周期选择器（日线 / 周线 / 月线）切换不同时间粒度。周线和月线数据由日线数据聚合而成：
- **日线**：原始每日 OHLCV
- **周线**：按周聚合（周一开盘、周五收盘、周内最高/最低）
- **月线**：按月聚合

### 2.3 图表交互

| 操作 | 效果 |
|------|------|
| 鼠标滚轮 | 缩放时间范围 |
| 拖拽选择区域 | 缩放到选定区间 |
| 悬停 K 线 | 显示当日 OHLCV 数据 |
| 悬停买卖点 | 显示交易详情卡片（价格、数量、原因） |

### 2.4 均线叠加与控制面板

图表默认显示 MA5（黄）、MA10（橙）、MA20（红）、MA30（绿）四条均线。均线数据在后端计算并随 K 线数据一同返回。

**控制面板**：图表左上角的控制面板提供复选框，可独立切换以下元素的显示/隐藏：
- **K线**：蜡烛图（默认显示）
- **MA5 / MA10 / MA20 / MA30**：四条均线（默认全部显示）
- **SAR**：抛物线转向散点叠加（默认隐藏）

切换时图表即时响应，右下角弹出简短提示。主图 y 轴显示虚线网格（splitLine），方便对齐价格。

### 2.5 成交量副图

K 线图下方独立显示成交量柱状图，红柱表示收盘 ≥ 开盘，绿柱表示收盘 < 开盘，附带 5 日和 10 日成交量均线。

### 2.6 副图技术指标

个股详情页和买卖点成交图页均支持通过工具栏按钮切换副图指标，包括：成交量 VOL、MACD、RSI、KDJ、布林带 BOLL、ATR 通道、CCI、威廉 %R、OBV 能量潮、ROC 变动率、趋势强度。所有指标基于连续日历日序列计算（消除周末缺口），图表仅显示交易日。副图支持折叠/展开，dataZoom 与主图联动，十字光标同步。

### 2.7 个股资料弹窗

在个股详情页或 K 线买卖点成交图页，点击「📊 个股资料」按钮可查看当前股票的财务数据、行业信息和概念题材。

**显示内容：**
- PE（市盈率 TTM）
- PB（市净率）
- ROE（净资产收益率 %）
- 总市值（亿元）
- 净利润（亿元）
- 流通股本（亿股）
- 数据更新日期
- 概念题材标签

弹窗居中显示，点击遮罩层或关闭按钮即可关闭。数据通过异步加载，加载中显示提示信息。如果当前股票无财务数据，弹窗会显示相应错误提示。

## 3. 策略工厂

策略工厂提供可视化的策略构建方式，无需编写代码。

### 3.1 基本操作

**创建策略：**
1. 点击「📋 策略工厂」进入策略构建页
2. 点击「+ 添加条件」打开卡片选择器
3. 从 25 种策略卡片中选择，设置买卖方向
4. 配置卡片参数（如 MA 快线=5、慢线=20）
5. 点击「▶ 运行回测」

**管理卡片：**
- **拖拽排序**：拖动卡片左侧的 ≡ 手柄调整顺序
- **编辑**：点击 ✏️ 按钮修改参数
- **删除**：点击 🗑️ 按钮移除卡片
- **上移/下移**：点击 ⬆/⬇ 按钮调整位置

### 3.2 可用卡片类型

|------|------|--------|

| 初始资金 | 回测起始资金（元） | 1,000,000 |
| 开始日期 | 回测起始日 | 2025-01-01 |
| 结束日期 | 回测结束日 | 当天 |
| 成交价模式 | close/next_open/half_spread | close |
| 佣金费率 | 每笔交易佣金比例 | 0.0003 |
| 印花税率 | 卖出时印花税比例 | 0.001 |
| 滑点成本类型 | 百分比 / 固定点数 | 百分比 |
| 滑点成本值 | 滑点数值 | 0.1 |

**指数快速填仓**：点击「快速填仓」按钮，可从沪深 300、中证 500 等指数一键导入成分股。

### 3.4 保存与加载

- **保存**：点击「💾 保存」按钮，输入策略名称，策略将以 JSON 格式保存
- **加载**：点击「📂 我的策略」查看已保存列表，选择加载
- **新建**：点击「📄 新建」清空当前状态
- **预览代码**：点击「📝 预览代码」查看由卡片生成的 Python 策略代码

### 3.5 内置模板

点击「📋 模板」可从 6 套预设策略中选择：

1. **双均线策略**：MA5/MA20 金叉买入、死叉卖出
2. **RSI 策略**：RSI<30 买入、RSI>70 卖出，含止损止盈
3. **MACD+KDJ 双确认**：两个指标同时金叉/死叉
4. **RSI+布林带**：RSI 超卖且突破布林下轨
5. **成交量突破**：放量且均线金叉
6. **趋势跟踪**：均线多头排列且 MACD 金叉

### 3.6 对比回测

对比回测功能可同时运行策略的多个参数变体，直观比较不同参数组合的绩效差异。

**进入对比模式：**
1. 在策略工厂页面配置好卡片（至少一张）
2. 点击「🔬 对比回测」按钮（黄色边框）
3. 弹出对比回测配置弹窗

**股票池设置（支持多股组合）：**
- 在"股票池"文本框中输入股票代码，每行一个或用逗号分隔（如 `000001,000858,600519`）
- 输入多只股票时，系统自动使用**共享资金池**模式，对所有股票统一执行策略，计算组合权益曲线和绩效
- 输入单只股票时使用单股模式
- 点击「快速填仓」按钮可一键导入指数成分股（沪深300、中证500、中证1000、创业板、科创50）

**配置变体：**
- 弹窗自动从卡片中提取所有数值型参数（如快线周期、慢线周期等）
- 每个变体使用中文标签显示参数，格式为"卡片标签 — 参数名"
- 修改参数值即可创建不同变体（初始默认值与主策略一致）
- 可编辑变体名称便于识别

**共享参数：**
弹窗顶部显示所有变体共享的参数，支持修改：
- 股票代码（支持模糊搜索）
- 回测起止日期
- 初始资金
- 成交价模式（close / next_open / half_spread）
- 佣金费率、印花税率
- 滑点成本类型和数值

**运行与查看结果：**
1. 点击「▶ 运行对比」按钮，所有变体通过多线程并行执行（最多 5 个并发）
2. 结果页面包含三个标签页：
   - **权益曲线对比**：ECharts 多系列折线图，不同颜色区分变体，含图例切换和面积填充
   - **指标对比**：表格列出所有变体的核心指标，最优值绿色高亮，最差值红色高亮
   - **交易信号**：下拉选择不同变体，查看对应的买卖信号明细

**导出变体代码：**
点击「📄 导出代码」可将所有变体的完整 Python 策略代码导出为文本文件，弹出系统原生保存对话框选择保存位置。可用于调试或备份。

**清除结果：**
点击「🗑 清除对比结果」回到策略详情页。

### 3.7 条件选股

条件选股功能允许使用策略卡片条件批量筛选股票，支持技术指标和财务指标的组合筛选。

**使用条件选股：**
1. 在策略工厂页面配置好筛选卡片（如 PE低于 20 且 ROE高于 15%）
2. 点击「🔍 条件选股」按钮
3. 设置股票池范围（可选择全市场或指定指数成分股）
4. 选择基准日期（默认为最新交易日）
5. 系统自动批量筛选，返回符合所有条件的股票列表

**支持的筛选条件：**
- 技术指标：MA 交叉、RSI、MACD、布林带、成交量等全部 21 种技术卡片，以及超级趋势、CMF、共振指标、趋势强度
- 财务指标：PE低于、PB低于、ROE高于
- 资金流向：主力资金净流入筛选
- 条件间使用 AND 逻辑（必须同时满足所有条件）

**股票池过滤：**
在条件选股页面可通过以下方式缩小筛选范围：
- 概念板块过滤：选择概念（如"人工智能"），支持 any（匹配任意）和 all（匹配所有）两种模式
- 行业过滤：按一级行业筛选（如"银行"）
- 市值过滤：按总市值区间筛选（单位：亿元）
- 流通股本过滤：按流通股本区间筛选（单位：亿股）

**验证单只股票：**
在条件选股页面可输入单只股票代码，测试其是否满足当前卡片条件，并查看具体原因说明（如"PE(TTM)=5.23 ≤ 20.00"）。

## 4. 代码编辑器

代码编辑器允许直接编写 Python 策略代码，适合高级用户。

### 4.1 编写策略

1. 点击「💻 代码编辑器」进入编辑器页面
2. 在文本框中编写 Python 代码
3. 代码必须包含以下两个函数：
   - `initialize(context)` — 初始化策略参数
   - `handle_bar(context, bar_dict)` — 每个 bar 执行的主逻辑

### 4.2 内置 API

代码编辑器支持和策略工厂完全相同的 API：

```python
def initialize(context):
    """设置策略参数，在回测开始时调用一次"""
    context.fast = 5
    context.slow = 20
    context.stock = "STOCK_CODE_PLACEHOLDER"  # 占位符，运行时会替换

def handle_bar(context, bar_dict):
    """每个交易日调用一次"""
    stock = context.stock

    # 获取历史数据
    fast_arr = history_bars(stock, context.fast + 1, '1d', 'close')
    slow_arr = history_bars(stock, context.slow + 1, '1d', 'close')

    if len(fast_arr) < context.fast + 1 or len(slow_arr) < context.slow + 1:
        return

    fast_ma = fast_arr[-context.fast:].mean()
    slow_ma = slow_arr[-context.slow:].mean()
    prev_fast = fast_arr[-context.fast-1:-1].mean()
    prev_slow = slow_arr[-context.slow-1:-1].mean()

    # 金叉买入
    if prev_fast <= prev_slow and fast_ma > slow_ma:
        order_target_percent(stock, 1.0)
        log.info("金叉买入信号")

    # 死叉卖出
    if prev_fast >= prev_slow and fast_ma < slow_ma:
        order_target_percent(stock, 0)
        log.info("死叉卖出信号")
```

### 4.3 股票占位符

使用 `"STOCK_CODE_PLACEHOLDER"` 作为股票代码占位符。回测运行时会自动替换为实际代码。

### 4.4 运行回测

点击「▶ 运行回测」，配置参数后执行。多只股票（逗号分隔）将触发多股组合回测。

### 4.5 API 帮助

点击「📖 API 帮助」按钮可查看完整的策略 API 文档。

## 5. 回测报告

### 5.1 查看报告

回测完成后，点击「📊 查看详情」进入报告页面。报告包含 3 个标签页：

> **数据完整性：** 回测前会自动检查数据库中该股票的实际交易日数量。若数据不足，会明确提示「请先更新数据后再回测」，不会静默使用模拟数据。因此，首次使用某股票前请先触发数据更新。

**权益曲线**
- 显示回测期间账户权益变化的折线图
- 可缩放查看任意时间段
- 图表含渐变填充区域

**交易信号**
- 列出所有买卖信号：日期、股票、方向、价格、数量（股）、市值
- 可同步单个信号到模拟盘（点击按钮）

**股票绩效**（多股回测）
- 每只股票的独立收益、交易次数、胜率
- 基于 FIFO（先进先出）配对计算

### 5.2 绩效指标

| 指标 | 说明 |
|------|------|
| 累计收益率 | 回测期间总收益率 |
| 年化收益率 | 折算为年化的收益率 |
| 最大回撤 | 账户权益的最大回撤幅度 |
| 夏普比率 | 风险调整后收益 |
| 胜率 | 盈利交易占比 |
| 交易次数 | 总交易笔数 |

### 5.3 导出报告

点击「导出报告」按钮，可选择保存为：
- **Excel (.xlsx)**：包含绩效指标、权益曲线数据、交易信号、股票绩效 4 个工作表
- **PDF (.pdf)**：包含权益曲线图表、绩效指标表格、交易信号表格

文件保存到用户选择的目录。导出的文件会先在 `temp_reports/` 生成，再复制到目标位置。

**可导出页面：**
- **策略详情页**（单股/多股回测结果）：包含完整的权益曲线和股票绩效
- **对比回测结果页**：导出当前选中的变体数据，支持多策略变体的独立报告

## 7. 数据更新

### 7.1 自动更新

系统默认在每日 **18:00** 自动检查并更新日线数据，每 **6 小时** 检查并更新财务数据。更新过程在独立子进程中运行，不会阻塞主界面。

### 7.2 手动更新

进入「⚙️ 设置」页面，点击「手动更新数据」按钮。更新进度会在控制台输出。

### 7.3 日线更新机制

- **增量更新**：每只股票从数据库中最新日期开始，仅下载缺失的交易日数据
- **失败跳过**：连续 3 次失败的股票自动跳过 30 天（如已退市股票）
- **锁机制**：同一时间只允许一个更新进程运行
- **速率控制**：每次请求间隔 0.5 秒，避免被 Baostock 限流

### 7.4 财务数据更新

财务数据更新器定期从 Baostock 获取最新财务指标：
- **更新内容**：PE(TTM)、PB、ROE、营业收入、净利润、流通股本
- **更新策略**：仅更新近一年有交易的活跃股票，自动过滤已退市股
- **增量触发**：超过 90 天未更新的股票或新上市的股票
- **市值计算**：更新完成后自动基于最新收盘价 × 流通股本计算总市值
- **手动触发**：在设置页面可单独触发财务数据更新（预计 3-10 分钟）

### 7.5 概念板块更新

概念板块数据更新器定期同步最新概念板块信息（如"人工智能"、"芯片"等），并维护股票与概念的关联关系。

### 7.6 资金流向数据

资金流向数据更新器定期从东方财富获取 A 股主力资金流向数据并存入数据库，支持历史查询和趋势分析。

### 7.7 新股同步

如需更新股票列表（处理新上市股票）：
```bash
python backend/sync_new_stocks.py
```

## 9. 日志系统

### 9.1 日志级别

回测日志使用颜色区分级别：

| 级别 | 颜色 | 说明 |
|------|------|------|
| INFO | 灰蓝 #9aa9cc | 一般信息 |
| WARN | 黄色 #f2c94c | 警告信息 |
| ERROR | 红色 #ff4c4c | 错误信息 |
| SUCCESS | 绿色 #4cff4c | 成功操作 |

### 9.2 筛选日志

日志区域上方的筛选按钮可快速过滤：
- **全部**：显示所有日志
- **信息**：仅显示 INFO 级别
- **警告**：仅显示 WARN 级别
- **错误**：仅显示 ERROR 级别
- **成功**：仅显示 SUCCESS 级别

### 9.3 导出日志

点击「📄 导出日志」按钮，将所有日志（不受筛选影响）导出为 `.txt` 文件。每行格式：

```
[2026-05-20 14:35:21] [INFO] 开始回测 000001
[2026-05-20 14:35:22] [WARN] 数据不足，跳过
[2026-05-20 14:35:23] [ERROR] 资金不足
[2026-05-20 14:35:24] [SUCCESS] 回测完成
```

### 9.4 日志限制

最多保留 **500 条**日志，超出后自动删除最旧的条目。用户向上滚动查看历史日志时，新日志不会触发自动滚底。

### 9.5 折叠/展开

策略工厂页的日志区域支持折叠：点击「🔼 折叠」隐藏日志，再次点击「🔽 展开」显示。

## 10. 实时模拟交易

### 10.1 进入页面

点击左侧导航的「⚡ 实时模拟」进入多股实时模拟交易页面。

### 10.2 配置策略

在控制面板中设置以下参数：

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 策略名称 | 便于识别的名称 | 空 |
| 初始资金 | 模拟交易起始资金（元） | 100,000 |
| 轮询间隔 | 行情轮询间隔（秒），自动根据股票数量调整 | 3~8 |
| 股票池 | 股票代码列表，逗号/换行分隔，最多 30 只 | 空 |
| 策略代码 | 完整的 Python 策略代码（含 initialize 和 handle_bar） | 空 |

**高级设置**（点击「⚙ 高级设置」展开）：
- 自定义轮询间隔（设为 0 则自动调整）
- 佣金费率、印花税率
- 滑点类型（百分比 / 固定点数）和滑点值

### 10.3 从策略工厂导入

在策略工厂页面配置并回测后，可直接点击「▶ 实时模拟」按钮将当前策略代码和参数一键导入实时模拟页面。

### 10.4 启动与停止

点击「▶ 启动多股策略」按钮，系统将在后台开启独立线程：
1. 批量获取股票实时行情
2. 每只股票独立调用 `handle_bar`
3. 订单按"先卖后买"顺序执行
4. 所有交易自动同步到模拟盘

点击「⏹ 停止策略」停止引擎。应用关闭时配置自动保存，下次启动时可选择恢复。

### 10.5 监控运行状态

页面实时展示：
- **持仓行情表格**：每只持仓股票的现价、涨跌幅、成交量
- **交易信号表格**：按时间倒序显示所有买卖信号
- **模拟持仓面板**：总资产、可用资金、持仓市值、浮动盈亏
- **运行日志**：引擎运行详情、下单确认、T+1 限制提示

### 10.6 T+1 交易规则

引擎遵循 A 股 T+1 交易规则：
- 当天买入的股票当日不可卖出
- 卖出时会自动减去当天买入量，只卖出"可卖数量"
- 跨日自动清空 T+1 锁定记录

### 10.7 页面恢复

如果关闭页面后重新进入，系统会：
1. 检测后台引擎是否仍在运行
2. 自动加载最新信号、持仓和行情
3. 从持久化配置恢复表单参数

## 11. 资金流向分析

### 11.1 查看资金流向

在个股详情页或 K 线图页，可查看当前股票的主力资金流向数据。

**数据内容：**
- 主力净流入（万元）
- 超大单、大单、中单、小单净流入
- 数据来源（东方财富 / 同花顺）

### 11.2 智能建议

系统基于当日主力净额和近 5 日历史趋势自动生成中文建议，例如：
- "主力净流入1234.56万，大额资金进场，建议关注"
- "主力连续3日净流出（含今日），累计-5000万，注意风险"
- "主力小幅净流入，建议观望"

建议规则覆盖：大额进出场、连续同向、小幅波动、历史趋势组合等场景。

### 11.3 批量查询

在持仓页面可批量获取所有持仓股票的资金流向，方便横向对比。
