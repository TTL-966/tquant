# 测试与验证

## 测试工具
- `python app/test_backtest.py` – 后端回测引擎独立测试
- 浏览器 F12 控制台 – 前端 JS 错误检查
- 手动点击测试 – 各页面功能验证

## 测试流程

### 1. 后端回测测试
```bash
python app/test_backtest.py
预期：状态 success，信号数量 > 50，权益曲线长度 > 1000。

2. 前端冒烟测试
启动应用后，依次点击每个导航项：

个人中心 – 显示持仓/交易记录/账户概况/每日收益

历史回测 – 显示策略模板列表，可加载

策略工厂 – 可添加卡片、编辑参数、运行回测、预览代码

策略详情 – 显示权益曲线、指标卡片、信号表格

买卖点成交图 – 显示 K 线+均线，可加载信号、切换股票

个股详情 – 可搜索（代码/名称）、显示 K 线和行业信息

API 文档 – 显示文档内容

设置说明 – 显示说明文字

3. 策略工厂专项测试
新建空白策略 → 添加各类卡片 → 编辑参数 → 保存 → 刷新后加载 → 确认卡片状态一致

加载每套模板 → 运行回测 → 确认产生信号

修改仓位比例 → 运行回测 → 确认交易手数变化

添加止损止盈 → 运行回测 → 日志中出现止损/止盈触发信息

4. 买卖点成交图专项测试

## 使用方式

在 Claude Code 中，输入 `@skill名` 即可激活对应技能。例如：
@fix-ui 成交价下拉框无限拉伸
@add-feature 添加一个“成交量缩量”条件卡片
@fix-backtest 回测无信号，检查代码生成
@perf-opt 个股详情页加载缓慢怎么优化
@run-tests 运行完整测试流程
